//Question 1 >> Installing node latest version and npm

Updating node
Clear NPM's cache:
sudo npm cache clean -f

Install a little helper called 'n'
sudo npm install -g n

Install latest stable Node.js version
sudo n stable

Updating npm
npm install latest-version
